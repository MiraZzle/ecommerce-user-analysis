Analza uivatel ecommerce pro doporuovn produkt

Autor: Matj Foukal

V tto statistick prci se budeme zabvat analzou dat zskanch od uivatel ecommerece portlu. Stanovm si hypotzy, kter nsledn na datech otestujeme, a tak prozkoumme korelace mezi jednotlivmi atributy nashromdnch o uivatelch. Takto zskan informace by v praxi typicky vedly ke stanoven lepch doporuovacch a makretingovch strategi nmi zkoumanho eshopu.

Pvod a struktura dat

Data byla nashromdna od 1000 uivatel v prbhu 2 msc zakladateli startupov spolenosti. Byla shromdna pro ely vzkumu a implementace personalizanch strategi v ecommerce. Cel dataset je veejn dostupn na strnkch kaggle.com.

Testovn hypotz

Otestujeme nsledujc hypotzy:

1. Vliv lokality uivatel (suburban, rural, urban) na hodnotu objednvky:

  • Nulov hypotza (H0): Prmrn hodnota objednvky je stejn pro vechny ti lokality.
  • Alternativn hypotza (H1): Prmrn hodnota objednvky se li mezi lokalitami.
In[]:
# Funkce pro proveden t-testu mezi dvma lokalitami
def t_test_location(data, location1, location2):
    # Vbr dat pro jednotliv lokality
    group1 = data[data["Location"] == location1]["Average_Order_Value"]
    group2 = data[data["Location"] == location2]["Average_Order_Value"]

    # T-test
    t_stat, p_value = stats.ttest_ind(group1, group2)
    return t_stat, p_value


# Prov t-testy
locations = ["Suburban", "Rural", "Urban"]
alpha = 0.05

# Proveden t-test pro vechny dvojice lokalit
for i in range(len(locations)):
    for j in range(i + 1, len(locations)):
        loc1, loc2 = locations[i], locations[j]
        t_stat, p_value = t_test_location(data, loc1, loc2)
        print(
            f"T-test mezi {loc1} a {loc2}: t-statistic = {t_stat:.4f}, p-value = {p_value:.4f}"
        )
        if p_value < alpha:
            print(f"Pro lokality '{loc1}' a '{loc2}' zamtme nulovou hypotzu (H0).")
        else:
            print(
                f"Nulov hypotza (H0) pro lokality '{loc1}' a '{loc2}' nen zamtnuta."
            )

# Vizualizace
plt.figure(figsize=(10, 6))
sns.boxplot(x="Location", y="Average_Order_Value", data=data)
plt.title("Vliv lokality na hodnotu objednvky")
plt.xlabel("Lokalita")
plt.ylabel("Prmrn hodnota objednvky")
plt.show()

2. Vliv zjm na frekvenci nkup:

  • Nulov hypotza (H0) Zjmy uivatel nemaj vliv na frekvenci nkup.
  • Alternativn hypotza (H1): Zjmy uivatel maj vliv na frekvenci nkup.

# Vytvoen binrnch promnnch pro zjmy
interests_dummies = data["Interests"].str.get_dummies(sep=",")

# Pidme frekvenci nkup k binrnm promnnm
interests_dummies["Purchase_Frequency"] = data["Purchase_Frequency"]


# Funkce pro proveden t-testu pro zjem
def t_test_interest(interests_dummies, interest):
    group1 = interests_dummies[interests_dummies[interest] == 1]["Purchase_Frequency"]
    group2 = interests_dummies[interests_dummies[interest] == 0]["Purchase_Frequency"]
    t_stat, p_value = stats.ttest_ind(group1, group2)
    return t_stat, p_value


# Testovn hypotzy pro kad zjem
alpha = 0.05
for interest in interests_dummies.columns[:-1]:
    t_stat, p_value = t_test_interest(interests_dummies, interest)
    print(
        f"Zjem: {interest} - T-test: t-statistic = {t_stat:.4f}, p-value = {p_value:.4f}"
    )
    if p_value < alpha:
        print(f"Pro zjem '{interest}' zamtme nulovou hypotzu (H0).")
    else:
        print(
            f"Nulov hypotza (H0) pro zjem '{interest}' nen zamtnuta."
        )
